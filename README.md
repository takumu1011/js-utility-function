# js-utility-function

<br>

## Feature

オブジェクト操作に便利な関数

<br>

## Using

```javascript
// import.
import Util from 'js-scroll-effect-module';
```

### Basic Use

```html
<script src="./js-utility-function.js"></script>

<script>
  Util.has(obj, key); //オブジェクトがそのプロパティを持っているかを返す
  Util.size(obj); //オブジェクトのプロパティ数を返す
  Util.toMap(obj); //オブジェクトをMapに変換
</script>
```
